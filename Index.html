IN1007 GameProject:

My second programming project, concluding the 2nd semester Programming Java at City (2021). This IntelliJ project demonstrates a comprehensive understanding of the use of CityPhysicsEngine. I created a game which has 3 different levels. The purpouse of the game is to collect enough gold amount on each level, so you can use portal to get to the next level. Details of each level are stored in each level class. The game has music, and each level has unique background. I made gold sound implementation, whenever gold is picked up, it makes gold gathering sound.
